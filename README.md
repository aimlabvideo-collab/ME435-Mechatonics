# ME 435 — Mechatronics and Measurement Systems

Course website for ME 435 at California State University, Northridge.

**Live site:** https://aimlabvideo-collab.github.io/ME435-Mechatonics/

This repository is published with GitHub Pages (Jekyll + the `just-the-docs` theme).

## Editing

- Pages are plain Markdown (`.md`). Edit them on GitHub (✏️ button) or locally.
- After editing locally, run:
  ```
  git add .
  git commit -m "Update notes"
  git push
  ```
  The site rebuilds automatically within 1–2 minutes.

## Structure

```
index.md          Home page
schedule.md       Weekly schedule
syllabus.md       Syllabus
resources.md      Software & references
chapters/         Per-chapter notes (ch1–ch7)
slides/           Lecture slide PDFs
examples/         Example problems
simulation/       MATLAB code
```
