# Simulation

MATLAB code used in lecture, organized by chapter.

| Folder | Topic |
|--------|-------|
| [`chapter3/`](chapter3/) | DC motor step response (state-space, `ode45`) |

Each chapter folder has its own README with details and how to run it.

## Getting the code

Download individual `.m` files from GitHub, or clone the whole course repo:

```bash
git clone https://github.com/aimlabvideo-collab/ME435-Mechatonics.git
cd ME435-Mechatonics/simulation
```

Open the `.m` files in MATLAB (uses built-in `ode45` — no extra toolboxes).
